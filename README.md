# C Algorithms

Academic project to apply data structure concepts and in a future GC and runtimes varieties.


## Algorithms

- Queue static.
- Stack statuc.
- List singly linked static.
- Queue dynamic.
- Stack dynamic.
- List singly linked dynamic.
- List doubly linked dynamic.


## Usage

To compile

	$ make
	
To clean

	$ make clean
